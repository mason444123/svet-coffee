# Свет. Coffee Rituals

Минималистичный сайт кофейни, экспортированный из Google Stitch и подготовленный для самостоятельного размещения.

## Текущий статус

- Адаптивный одностраничный сайт
- Локальные изображения без зависимости от временных Stitch URL
- Подготовлена форма заказа: имя, телефон и адрес доставки
- Отправка отключена до подключения безопасного backend-обработчика Telegram-бота

## Локальный запуск

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Подключение Telegram-бота

Токен Telegram-бота нельзя размещать в `index.html`: он станет публичным. На следующем этапе форма будет отправлять данные на серверный endpoint, а уже сервер — в Telegram Bot API.
