# Свет. Coffee Rituals

Минималистичный сайт кофейни, экспортированный из Google Stitch и подготовленный для самостоятельного размещения.

## Текущий статус

- Адаптивный одностраничный сайт
- Локальные изображения без зависимости от временных Stitch URL
- Заказ через Telegram обозначен как будущая функция
- Формы и серверного хранения данных сейчас нет

## Локальный запуск

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Подключение Telegram-бота

Когда бот будет готов, замените неактивную кнопку в секции `#contact` на ссылку:

```html
<a href="https://t.me/USERNAME" target="_blank" rel="noopener">Заказать в Telegram</a>
```
